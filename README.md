# Course Management System

## Overview
A web-based Course Management System built using a monolithic architecture.

The system allows:
- Admins to manage users, departments, and courses
- Teachers to manage assignments, attendance, and grading
- Students to enroll in courses, submit assignments, and track academic progress

---

## Tech Stack

### Backend
- Java

- Spring Security
