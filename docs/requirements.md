# Course Management System

## Purpose
A web-based application where:
- Admin manages users, departments, and courses
- Teachers manage assignments, attendance, and grading
- Students enroll in courses, submit assignments, and track academic progress

---

## Roles

### Admin
- Manage users
- Manage departments
- Manage courses
- Assign teachers to courses

### Teacher
- Create and manage courses
- Create assignments
- View student submissions
- Grade assignments
- Mark attendance
- View enrolled students

### Student
- Enroll in courses
- View course details
- Submit assignments
- View grades
- View attendance
